# Personal Portfolio Website
## Overview
This project is a responsive personal portfolio website developed using HTML, CSS, and JavaScript. The website is designed to showcase my skills, projects, and contact information in a professional and visually appealing manner.
The portfolio provides an interactive user experience with smooth navigation, responsive layouts, hover effects, and modern UI design. It serves as a digital profile to represent my journey as an aspiring Full Stack Developer.

## Features

- Responsive and modern website design
- Smooth scrolling navigation
- Attractive hero/home section
- About Me section with personal introduction
- Skills section with technical skills display
- Projects section showcasing project details
- Contact form for communication
- Interactive hover animations and effects
- Mobile-friendly layout

## Technologies Used

- HTML5
- CSS3
- JavaScript

### Home

## Purpose of the Project

The main purpose of this project is to practice front-end web development skills and create a professional online presence. This project also helped in understanding responsive design, UI styling, and basic JavaScript functionality.

This portfolio website reflects my passion for coding and web development. It demonstrates my ability to design and develop responsive websites using front-end technologies while continuously improving my technical and creative skills.
